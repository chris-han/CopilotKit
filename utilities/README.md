# Utilities

This directory contains utility tools and scripts for various purposes.

## Available Utilities

### Tax Newsletter Generator

**Directory:** `tax-newsletter/`

Automatically generates weekly newsletters with the latest updates on US-China cross-border tax regulations.

**Features:**
- Monitors official IRS and China tax administration sources
- Tracks professional tax service updates
- Focuses on cross-border taxation topics (FATCA, CRS, tax treaties, etc.)
- Generates markdown formatted newsletters
- Automated via GitHub Actions (runs weekly)

**Documentation:** See [tax-newsletter/README.md](./tax-newsletter/README.md)

**Generated Newsletters:** Available in `/newsletters/` directory

**GitHub Action:** `.github/workflows/tax_newsletter.yml`
